Bannere

Bine aţi venit pe Meet Firefox!

Meet Firefox este sait de promovare pentru web browser-ul Mozilla Firefox. Aici poţi să aflii mai multe informaţii despre acest navigator şi să-l descarci.

Functionalitati noi

Internetul inseamna inovare si Firefox 3 tine pasul cu zeci de noi functionalitati incluzand un "smart location bar", marcare cu un singur clic si navigare foarte rapida.

Securitate

Pentru Mozilla Firefox securitatea online este o prioritate. Acest browser include masuri de protejare anti-phishing si anti-malware. Datorita faptului ca acest navigator este open source, foarte multi experti se ocupa de securitatea acestuia.

Productivitate

Firefox are foarte multe functionalitati care ajuta sa lucrezi rapid. Acesta include "spell checking", restaurarea seziunii, "full zoom", si multe altele.

Modificare

Pentru ca toata lume foloseste internetul diferit, Firefox are mii de add-on-uri pentru a lucra cum vrei tu.
